Privacy Policy

Last updated: August 2025

1. Introduction

CommandBar+ ("we," "our," or "us") is committed to protecting your privacy. This Privacy Policy explains how we collect, use, disclose, and safeguard your information when you use our browser extension and related services.

2. Information We Collect

2.1 Information You Provide

We do not store any information from you. All data is stored locally on your device. This includes:

  • Appearance settings (dark/light mode)
  • To-do list items
  • Blocked websites

2.2 Automatically Collected Information

CommandBar+ uses the following information to provide its services:

  • Tabs and browsing activity
  • Active tab information
  • Context menu interactions
  • Window and session data
  • Bookmarks and top sites
  • Downloads management
  • Extension management
  • Tab groups
  • Browsing data

3. How We Use Your Information

We use the information to provide and improve our commands such as:

  • List All Tabs
  • Ask Browser Commands
  • Manage Blocklist
  • Manage To-do List

4. Information Sharing and Disclosure

We do not sell, trade, or otherwise transfer your information to third parties, except in the following circumstances:

  • AI Commands Centered Around the Webpage (Summarize, Ask)
  • Shorten URL Command
  • Generate QR Code Command

None of the information sent to these 3rd parties is personal or sensitive.

5. Third-Party Services

CommandBar+ integrates with 3 third-party services. These services have their own privacy policies, and we encourage you to review them. We are not responsible for the privacy practices of third-party services:

  • Groq
  • TinyURL
  • QRServer

6. Remote Code Use

CommandBar+ does not use any remote code to provide its services. This code is hosted completely on your machine. We do not have access to the code and cannot modify it.

7. Permissions

When you download CommandBar+, we ask for a few permissions. If some of these are turned off, a few features might not work properly. We recommend that you choose the permissions you will need.

Tabs - The tabs permission is necessary for several commands, such as Auto Group Tabs, Ask Tabs, and List All Tabs.
ClipboardWrite - The output of certain commands can be quickly copied with a click if the clipboardWrite permission is satisfied.
Storage - Storage is necessary to store user settings, such as appearance (light/dark mode), width/height, tasks, and the blocklist.
ActiveTab - The activeTab is necessary for many commands, such as Copy Current URL, Summarize Page, Ask AI About Webpage, etc.
Scripting - CommandBar+ uses the scripting permission to programmatically insert elements into the DOM, further enhancing accessibility.
ContextMenu - The context menus permission is necessary so users can open CommandBar+ extremely quickly from the right click menu.
Bookmarks - Several commands require the bookmarks permission, like Ask Bookmarks, Add Current Page as Bookmark, etc.
Sessions - Access and manage browser data for several commands, like List All Windows.
Downloads - Several commands utilize the downloads permission, such as Ask Downloads.
Management - Allows CommandBar+ to query, enable, disable, or uninstall other extensions and apps installed in the browser.
History - Several commands, such as Ask History, use this command.
TabGroups - Manage tab groups with Auto Group Tabs and Delete Tab Groups Command.
TopSites - Used solely for the Ask Top Sites command.
BrowsingData - Allows CommandBar+ to clear browser history.
Notifications - Push notifications when the user tries to go against their blocklist.
Host Permissions - Used to interact and extract data from the webpage.

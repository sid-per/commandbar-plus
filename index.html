Setup & Configuration

Get started with CommandBar+ in just a few steps.

Step 1: Install from Chrome Web Store

Click "Add to Chrome" to install CommandBar+.

Step 2: Configure Groq API Key

Right-click the icon and select "Options" to add your Groq API key for AI features.

Step 3: Start Using Commands

Type in the search bar and use arrow keys or tab to pick a command. Press enter to run it.
